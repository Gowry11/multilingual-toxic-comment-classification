# Multilingual Toxic Comment Classification using XLM-RoBERTa

## Project Overview

This project fine-tunes the **XLM-RoBERTa** model for multilingual toxic comment classification using the Hugging Face Multilingual Toxicity Dataset.

The objective is to automatically identify **toxic** and **non-toxic** comments across multiple languages using a state-of-the-art transformer-based deep learning model.

---

## Technologies Used

* Python
* Hugging Face Transformers
* XLM-RoBERTa
* PyTorch
* Scikit-learn
* Gradio
* NumPy

---

## Features

* Multilingual text classification
* Transformer-based deep learning architecture
* Fine-tuning of XLM-RoBERTa for toxicity detection
* Model evaluation using Accuracy and F1-Score
* Real-time toxicity prediction
* Interactive Gradio web interface

**Multilingual Toxicity Dataset**

Dataset Source:

## Model

**Pretrained Model:** XLM-RoBERTa Base

Model Source:
https://huggingface.co/xlm-roberta-base

---

## Project Workflow

1. Load the multilingual toxicity dataset.
2. Perform data preprocessing and tokenization.
3. Fine-tune the XLM-RoBERTa model.
4. Evaluate model performance using classification metrics.
5. Generate predictions on new text inputs.
6. Deploy the prediction system using a Gradio interface.

---

The model performance is evaluated using:

* Accuracy Score
